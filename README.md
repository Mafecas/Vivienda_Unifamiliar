# Vivienda_Unifamiliar
Diseño arquitectónico completo de vivienda unifamiliar de 150m² en un piso. Incluye modelo 3D interactivo con recorrido interior, y 7 planos DXF para AutoCAD: arquitectónico, cimentación (zapatas aisladas), eléctrico (7 circuitos), agua fría/caliente, desagüe, gas y cubierta.
